SCENARIO DE BASE

git clone url: récupère un dépot d'une adresse url

git add fichier: : marque le fichier a récupere

git add* :marque tous les fichiers a consigner

git commit -m :consigne les fichiers a marqué dans le dépot

git status :affiche l état du dépot local

git push origin :envoi les modifications sur le dépot local
